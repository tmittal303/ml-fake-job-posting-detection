Project Overview:
This project builds a machine learning model to classify job postings as real or fake. The process starts with exploring the dataset, cleaning the data, handling missing values, and understanding patterns in different features.

What I Did:
Cleaned and prepared the dataset
Focused on important text fields like job descriptions, requirements, and company profiles
Converted text data into numbers using techniques like TF-IDF and CountVectorizer

Models Used:
Logistic Regression
Naive Bayes
The dataset was split into training and testing sets to evaluate performance properly.

Evaluation:
Instead of only checking accuracy, I used multiple metrics:
Precision
Recall
F1-score
This helped better understand how well each model performs.
Tuning model parameters
Final Result

The final system can take a new job posting and predict whether it is fraudulent or legitimate. I also analyzed how the model makes decisions and noted its limitations in real-world use.

Summary

This project shows the complete process of building a text classification model, from data cleaning to final prediction, in a simple and practical way.
